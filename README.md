## Getting Started

First, run the development server:

```bash
#install depedencies
npm i
#or
yarn

#development mode
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
#or other port
- yarn dev -p (your port) #ext yarn dev -p 8088
